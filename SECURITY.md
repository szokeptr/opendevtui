# Security Policy

OpenDevTUI runs local commands from workspace configuration and is intended for trusted development environments.

Do not expose a session running OpenDevTUI directly to the public internet. If you run it on a server, access it over SSH or behind strong private network controls.

To report a security issue, open a private GitHub security advisory for the repository if available. If advisories are not enabled, contact the maintainer directly through the GitHub profile linked from the repository.
